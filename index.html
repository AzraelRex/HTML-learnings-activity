In order to comment in HTML you need to use the < followed up by !-- then close the tag with the > symbol.

HTML Links Apparently in order to be able to link either a hyperlink, a video, an image or any location in general such as the frontpage of Google, all you have to do is use the a href tag followed by the link, and then a quick description about it.

HTML Forms This one was very tricky for me as well, as it has plenty of nuances and circumstances that you need to be aware of it, in order to fully use it to it's utmost potential.

In order to make a form in HTML, you need to first write a form tag, an input tag to define a single line of text, then there's the label tag, it has a label element which is used to define labels for various elements such as checkbox, color data etc.

The input type has three functions, one of them is the input type radio button, which serves to have the user be able to pick out various options to choose from. Then there is the input type checkbox which is used in order to display checkboxes for choices in HTML.

Then there's the input type submit, which is used to for a submit button which will submitted to a form handler, which is typically a file in a server with a script for processing input data.
